Downloading leptonlib pre-built binaries and header files for WindowsÂ¶

The easiest way to use leptonlib is to download the latest version of leptonica-1.xx-win32-lib-include-dirs.zip, the pre-built binary archive, from the Leptonica website.

It contains the entire lib and include directories needed to build Windows-based programs that use the static or dynamic versions of the leptonlib library (including static library versions of zlib, libpng, libjpeg, libtiff, and giflib). The libraries are built with Microsoft Visual Studio 2008 32-bit. The complete contents of the archive can be seen here.

Unpack this file to the same directory that leptonlib will be in. For example, after unpacking you should have something similar to this:

BuildFolder\
  include\
  leptonlib-1.xx\
  lib\

The archive contains the latest releases of the supported image libraries at the time leptonlib was released. For the current version (Leptonica-1.66) these are:

Image Library Versions
Library Version | Date
zlib 1.2.5 19-Apr-2010
libpng 1.4.3 25-Jun-2010
libjpeg 8b 16-May-2010
libtiff 3.9.4 15-Jun-2010
giflib 4.1.6 10-Nov-2007

A simple method to get the versions of leptonlib and its image libraries is shown in the Quickstart.

If you’d like to determine the compiler options used to build these binaries or build them yourself see (Optional) Building the leptonlib library and (Optional) Building zlib, libjpeg, libpng, libtiff and giflib.

Note

Executables for the programs in the leptonlib /prog directory are not included in this archive. Follow these instructions to build them yourself.
